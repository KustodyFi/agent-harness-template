# Harness — Pre-Task Loading

Before starting ANY task, follow the startup sequence in `AGENTS.md`.

For harness-governed tasks (anything under `.cowork/tasks/`), also read:

1. `.cowork/HARNESS_SPEC.md` — task-stage workflow rules
2. `.cowork/STATUS.md` — project dashboard
3. Active task's `STATE.yaml` — current stage and status

**Follow all rules strictly. No exceptions.**
